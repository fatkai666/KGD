# KG Design

高仿ANTD组件库

[学习视频路径](https://coding.imooc.com/class/428.html)

技术栈：React + Hooks + TS 



安装：

```
npm install kgd --save
```

或

```
yarn add kgd --save
```



组件：

Alert/AutoComplete/Button/Icon/Input/Menu/Select/Tabs/Upload

引入方式：

```
import { 组件名 } from 'kgd'
```

样式引入：

```
import "kgd/dist/index.css"
```
